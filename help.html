File Hash Verifier Help

Main Window

The main window provides access to all features of the File Hash Verifier:

  • Verify Single File - Calculate and verify hash of a single file
  • Compare Two Files - Compare hashes of two files
  • Batch Process - Process multiple files at once
  • View History - View previous hash operations
  • Export History - Export history to various formats
  • Change Log Directory - Set where log files are saved
  • Toggle Theme - Switch between light and dark themes

Verify Single File

This feature allows you to:

  1. Select a file to verify
  2. Choose a hash algorithm (MD5, SHA-1, SHA-256, SHA-512)
  3. View the calculated hash
  4. Verify against a known hash
  5. Save the hash to a file
  6. Copy the hash to clipboard
  7. Generate a hash file
Note: The hash is displayed in the last 10 digits for better readability in reports.

Compare Two Files

This feature allows you to:

  1. Select two files to compare
  2. Choose a hash algorithm
  3. View the hashes of both files
  4. See if the files are identical

Batch Process

This feature allows you to:

  1. Select multiple files to process
  2. Choose a hash algorithm
  3. Calculate hashes for all files
  4. Save results to a log file

View History

The history view shows:

  • Date and time of each operation
  • File name and path
  • Algorithm used
  • Hash value (last 10 digits)
  • Verification status

Export History

You can export history in multiple formats:

  • HTML - Formatted report with styling
  • PDF - Printer-friendly document
  • CSV - Spreadsheet-compatible format
  • JSON - Structured data format

Export options include:

  • Date range selection
  • Format selection
  • Custom save location

Settings

Log Directory

You can change where log files are saved:

  1. Click "Change Log Directory"
  2. Select a new directory
  3. The directory must be writable

Theme

Toggle between light and dark themes:

  • Light theme - Default, easy to read in bright environments
  • Dark theme - Easier on the eyes in low-light conditions
